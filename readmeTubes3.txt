Ringkasan spesifikasi:
1. merangkum semua topik yang pernah dibahas dalam bahasa Java dan memakai tools yang disarankan di kelas
2. menggunakan versi thread
3. boleh melakukan gamifikasi (buat spesifikasinya kalau dilakukan), dengan menggunakan secara maksimal kelas-kelas yang pernah dibuat
4. menggunakan design pattern
5. mengandung asersi dan exception
6. memakai API collection
7. setiap unit memiliki JUnit
8. aplikasi harus dimunculkan metriksnya (jdepend memperbaiki kesalahan)
9. boleh menggunakan cmd atau gui, kalau gui harus menerapkan model MVC
10. checkstyle (mengurangi error lagi sebisa mungkin)
